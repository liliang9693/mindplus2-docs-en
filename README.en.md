# MindPlus2 Documentation 

The source code of the documentation hosted at [www.mindplus.cc/](https://mindplus.cc/en.html)

## Host the documentation locally

1. **Install Software**

   - Python3
   - Git

2. **Install mkdocs and mkdocs-material**

   ```bash
   python -m pip install mkdocs mkdocs-material mkdocs-minify-plugin mkdocs-glightbox
   ```

3. **Clone the Repository**

   ```bash
   git clone  https://github.com/DFRobot/mindplus2-docs-en
   ```

4. **Navigate to the cloned repository's path** in the command line (where you can see the `mkdocs.yml` file), and run the local mkdocs test server:

   ```bash
     
   python -m mkdocs serve -a 127.0.0.1:8090
   
   ```

5. **Access the local test server** using a web browser by visiting [127.0.0.1:8090](http://127.0.0.1:8090/).
