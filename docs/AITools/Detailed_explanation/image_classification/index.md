# 4.2.1 图像分类

欢迎使用Mind+图像分类功能！本章节将介绍Mind+图像分类的使用方法。

## 章节概览

本章节包含以下内容：

- **快速体验**：适合初学者的简化操作模式
- **专业模式**：提供高级功能和精细控制的专业模式

## 快速体验模式

适合初学者、教学演示和快速原型验证。提供一键式操作和可视化界面，无需深入了解机器学习原理。

[📖 查看快速体验详细教程](quick_experience/quick-experience.md)

## 专业模式

适合有一定基础的用户和研究人员。提供精细控制、完整的数据管理系统和实时训练监控。

[📖 查看专业模式详细教程](professional_mode/professional-mode.md)

请根据需要选择相应的模式进行学习。
