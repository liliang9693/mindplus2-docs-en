# 3.1.3.1 Motion

Motion-related blocks are primarily used to control a character's movement and rotation—such as moving forward, changing coordinates, or rotating by a certain angle—to create dynamic effects on the stage.

| Blocks                                                                                                                      | Note                                                                                                                                        |
| --------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------- |

| ![image](./3131.assets/pic_1781491904658.png) | The character turns to the right by the specified angle.                                                                                    |
| ![image](./3131.assets/pic_1781491909968.png) | The character turns left by the specified angle.                                                                                            |
| ![image](./3131.assets/pic_1781491916233.png) | The character moves to a random location or to the position of the mouse pointer.                                                           |
| ![image](./3131.assets/pic_1781491922067.png) | The character moves to the specified coordinates.                                                                                           |
| ![image](./3131.assets/pic_1781491927714.png) | The character glides to a random location or the mouse pointer's position within a specified time.                                          |
| ![image](./3131.assets/pic_1781491932829.png) | The character glides to the specified coordinates within the designated time.                                                               |
| ![image](./3131.assets/pic_1781491940238.png) | The character is facing a specified direction.                                                                                              |
| ![image](./3131.assets/pic_1781491946088.png) | The character faces the direction of the mouse pointer.                                                                                     |
