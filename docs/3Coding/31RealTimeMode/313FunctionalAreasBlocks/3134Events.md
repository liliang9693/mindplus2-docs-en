# 3.1.3.4 Events

Event blocks are used to respond to program startup or external trigger behaviors, such as clicking flags, pressing keys, receiving broadcast messages, etc., to control when the program starts executing or responding to external operations.

| ![image](./3134Events.assets/pic_1780550528225.png) | When you press a key on the keyboard, you start executing each line of instruction blocks in order.                                                            |
| ![image](./3134Events.assets/pic_1780550534004.png) | When you tap a character, you start executing each line of command blocks in order.                                                                            |
