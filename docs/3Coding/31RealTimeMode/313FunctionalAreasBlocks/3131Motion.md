# 3.1.3.1 Motion

Motion-related blocks are primarily used to control a character's movement and rotation—such as moving forward, changing coordinates, or rotating by a certain angle—to create dynamic effects on the stage.

| blocks                                                                                                                      | Note                                                                                                                                        |
| --------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------- |

| ![image](./3131.assets/pic_1780541403722.png) | The character turns to the right by the specified angle.                                                                                    |
| ![image](./3131.assets/pic_1780541411368.png) | The character turns left by the specified angle.                                                                                            |
| ![image](./3131.assets/pic_1780541418730.png) | The character moves to a random location or to the position of the mouse pointer.                                                           |
| ![image](./3131.assets/pic_1780541427267.png) | The character moves to the specified coordinates.                                                                                           |
| ![image](./3131.assets/pic_1780541435688.png) | The character glides to a random location or the mouse pointer's position within a specified time.                                          |
