# 3.4.2 Settings

The Settings interface serves as the entry point for Mind+'s global settings. Click the "Settings" button to open it. The settings in Micropython Block Mode are identical to those in Real-time Mode, offering features such as language switching, theme selection, "Contact Us," and Developer Mode. We won't go into further detail here; please refer directly to the settings instructions for Real-time Mode.
