# 3.2.3.1 Control

Control blocks are used to implement logical control of program flow, such as conditional statements, loops, or wait operations, helping the program make decisions based on different situations and run in an orderly manner.

| Blocks                                                                                                                           | Note                                                                                                              |

| ![image](./3231Control.assets/pic_1780986600829.png) | The program below this module will not execute until the conditions are met.                                      |
| ![image](./3231Control.assets/pic_1780986417605.png) | Execute the program within the module 10 times.                                                                   |
| ![image](./3231Control.assets/pic_1780986652171.png) | Execute the program within the module until the condition is met, then exit the loop.                             |
| ![image](./3231Control.assets/pic_1780986426155.png) | Execute the program within the module until the condition is met, then exit the loop.                             |
| ![image](./3231Control.assets/pic_1780986437235.png) | Perform a logical evaluation of a single condition; if the condition is met, execute the program below.           |
| ![image](./3231Control.assets/pic_1780986444789.png) | Determine whether the condition is true; if true, execute the program above; if false, execute the program below. |
