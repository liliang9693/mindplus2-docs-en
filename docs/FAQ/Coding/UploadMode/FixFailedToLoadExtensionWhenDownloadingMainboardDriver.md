# Fix "Failed to Load Extension" When Downloading Mainboard Driver


What should I do if I get a "Failed to load extension" error when downloading the latest motherboard driver?


## Problem Description

When I click on the latest motherboard or extension in the extension library, I get a "Failed to load extension" message.


## Solution

* If your version of Mind+ is older than V2.0.6, please update to the latest version from the Mind+ official website first.
* If the version is V2.0.6 or later, check your network connection.
