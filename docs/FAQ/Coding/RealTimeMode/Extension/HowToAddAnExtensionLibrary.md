# How to Add an Extension Library


## Problem Description

* In Mind+ version 2.x, I can't find the option to load a user library via a link or locally.
* User libraries from Mind+ version 1.x cannot be found in Mind+ version 2.x


## Analysis of Causes

Version 2.x of Mind+ includes major updates to the user library import mechanism, and the way libraries are created has been revised.

* User databases from Mind+ version 1.x cannot be imported directly; they must be converted first.
* The feature to add custom user databases is currently under development and will be available in a future version.


## Solution

* The official team has updated all included user libraries to be compatible with Mind+ 2.x. You can search for and load them using the built-in library manager in Mind+ 2.x.
* If you have a Mind+ 1.x user database that needs to be converted, please send the relevant user database files—either a link to the Git repository or the MPExt file—to the official email address: `MindPlus@dfrobot.com` for conversion.

* If you need to add a motherboard or expansion module, please send the relevant materials [product information, library files, use cases, etc.] to the official email address: `MindPlus@dfrobot.com`.
