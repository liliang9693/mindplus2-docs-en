# **Fix "Please Connect to Mind+Link First" Error in Extension Library**


## Problem Description

In Mind+ version 2.x, when I open the Extension Library, the message “Please connect to Mind+Link first” appears in the upper-right corner of each extension card. What should I do?


## Analysis of Causes

* In Mind+ version 2.x, pages communicate with the extension library and hardware via the Mind+Link plugin. This message indicates that the connection to Mind+Link has failed.
* For the online version, you need to download and launch the Mind+Link software separately.

* The offline version comes with Mind+Link pre-installed and launches automatically when the software starts, so there is no need to install or launch it separately.


## Solution

Depending on whether you are currently using the “Online Version” (Mind+ opened in a browser) or the “Offline Version” (Mind+ opened as a desktop application), follow the troubleshooting steps below:


### Use the online version:

* If this is your first time using the app, return to the programming page, click “Connect Device” in the menu bar to download the Mind+Link software, install it, and launch it. Then refresh the page and click “Connect Link”;
* If Mind+Link is already installed, check to see if the Mind+Link tray icon is in the lower-right corner of the desktop; if not, launch Mind+Link;

* If Mind+Link is already running but the webpage still shows a connection prompt, try refreshing the page first. If you still cannot connect, check your computer's network settings. If a VPN is enabled, you'll need to turn it off first.
* If none of the above steps work, please provide feedback via the official website email.

### Using the Offline Version:

* Try restarting Mind+;
* If the problem persists or occurs frequently, try running Mind+ as an administrator;

* If none of the above steps work, please provide feedback via the official website email.
