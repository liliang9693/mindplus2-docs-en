# How to Fix "pyenv: command not found" Error

## Problem Description

When I try to switch Python environments using pyenv, I get a "command not found" error.

## Examples of the Phenomenon

![img](https://mindplus.dfrobot.com.cn/mp2/FAQ/AITools/img/pyenv_command_not_found.png)

## Solution

* First, confirm that the program is connected to the Xingkong M10 board and running (Mind+ top menu: Connect 10.1.2.3).
* [Next, make sure the UNIHIKER M10 system firmware is up to date (you can check the system version by long-pressing the Home menu > View System Information); it must be at least v0.4.1. To update the UNIHIKER system firmware: Click here to view](https://www.unihiker.com.cn/wiki/m10/burner)
