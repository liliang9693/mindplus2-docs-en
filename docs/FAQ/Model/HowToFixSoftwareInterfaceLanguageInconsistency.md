# How to Fix Software Interface Language Inconsistency



## Problem Description

Feedback from some users: The software’s main interface is in Chinese, but when they go to the “Create Model Training” page, the interface is displayed in English.


## Examples of the Phenomenon

* **Main Screen (Chinese)**

![img](https://mindplus.dfrobot.com.cn/mp2/FAQ/AITools/img/58068e29-ceb1-43a5-a827-3651833cc7ba.png)

* **Create a Model Training Page (English)**

![img](https://mindplus.dfrobot.com.cn/mp2/FAQ/AITools/img/13513942-ac75-43b0-8dfd-4616f991bf61.png)


## Solution

Method 1: Download and install the latest version of Mind+.

Method 2:

* Find the language selection menu in the upper-right corner of the main screen.
* Switch between "Simplified Chinese" and "English".
* After switching, return to the Create Model and Train page.

  ![img](https://mindplus.dfrobot.com.cn/mp2/FAQ/AITools/img/Model-shows-abnormalities.gif)
