**格式化输出**

| **指令**                                                     | **说明**                                                     |
| ------------------------------------------------------------ | ------------------------------------------------------------ |
| ![img](png_file/1761289587185-cea7b908-9698-4a7b-94a6-2b84c3b859c6.png) | 格式化字符串——%填充符表示法。例如：如果有一个变量 num=3.14159，使用 “Value: %.2f ”%num 会得到结果 “Value：3.14”。%.2f表示：将变量格式化为浮点数，并保留两位小数。 |
| ![img](png_file/1761289592901-ebe2c705-0daa-449b-9be8-caeda1592a98.png) | 格式化字符串——format填充符表示法。例如：“Value: {:.3}” format（5）会得到结果 “Value：5”。详细解释：“Value: {:.3}” format（5）由于传入的是整数，被当作字符串处理，精度 .3表示截取前3个字符，而 “5” 只有1个字符，所以输出结果就是 “Value：5”。 |
| ![img](png_file/1761289599941-a0042bc0-8870-4296-9210-430f150ffde1.png) | 文本转义符，算特殊字符类型。\n 表示换行符，换到下一行的开头。 \r 表示回车符，将光标定位到当前行的开头，不会跳到下一行。 \n\r 表示先换到下一行，并将光标定位到当前行的开头。 \r\n 表示先将光标定位到当前行的开头，然后从该位置换行。 \b 表示退格符号，类同键盘上Backspace键。 \t 表示制表符，将光标移动到下一个制表符的位置，类同在文档中使用键盘Tab键。 \\ 标识反斜杠字符，如需使用到‘\’,需使用双反斜杠。 |
