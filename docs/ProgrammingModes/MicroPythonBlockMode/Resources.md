在MicroPython积木模式中，“资源文件”是指用于 存放项目中需要被程序调用的各种辅助文件的区域。它的作用就像一个“项目素材库”，方便你在编写 MicroPython 程序时引用这些资源。  

![img](png_file/1761529843254-7361a4c9-507c-41b7-96ad-2b39cbde3ed8.png)

其中，资源文件可以包含的内容如下：

| **资源类型**            | **示例**           | **用途**                     |
| ----------------------- | ------------------ | ---------------------------- |
| 图片文件（.png / .jpg） | 角色图片、图标     | 程序中显示图像或作为界面素材 |
| 音频文件（.mp3 / .wav） | 背景音乐、提示音   | 用于播放音效或者语音提示     |
| 文本文件（.txt）        | 数据记录、对白文本 | 用程序读取作为数据或内容     |
| JSON / CSV 文件         | 配置数据、表格数据 | 用于数据分析或参数读取       |
| Python脚本（.py）       | 自定义库代码       | 可被主程序导入复用功能       |

 点击“上传文件”按钮，可从本地添加所需的资源文件。上传后的图片类资源支持直接预览，方便快速确认内容是否正确。
