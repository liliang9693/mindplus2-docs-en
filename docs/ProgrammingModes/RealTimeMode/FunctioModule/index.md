# 3.1.3 功能区-模块
功能区是实时模式中操作的重要区域，功能区将程序操作、视觉效果与声音控制击中在一起，方便用户快速进行创作与调试，主要包括模块、造型、声音。

![img](png_file/1760695258072-cdaa0d3c-4388-41d7-ba3b-f607647fafaf.png)

#### 1. 模块

在实时模式下，模块区的积木根据功能被划分为 9 个类别：运动、外观、声音、事件、控制、侦测、运算符、变量和函数， 这样的分类方式有助于用户快速定位所需积木并构建完整的程序结构。
